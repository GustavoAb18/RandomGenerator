# RandomGenerator

* Montei um código simples em Ruby, onde gera variedades de letras, números e caracteres aleatórios.

---

## Rodando o programa

* Segue esse passo a passo para poder rodar o programa:
    1. Vá no site [RubyInstaller](https://rubyinstaller.org/downloads/) e baixe a última versão disponível para Download, e segue o passo mostrado no instalador;
    2. Depois, abre um terminal e digite: `ruby -v` para verificar se está instalado. Isso mostrará uma versão do ruby, exemplo: 'ruby 3.4.8'.
    3. Logo em seguida, caminhe no terminal onde você baixou o código.
    4. Por fim, execute o código: 'ruby rubyName.rb' para executar o código.
